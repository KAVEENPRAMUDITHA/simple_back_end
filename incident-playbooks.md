# Incident Playbooks

## Common Issues

### Server Down

**Immediate Actions:**

1. Check server status.
2. Restart server.
3. Escalate to DevOps team.

### DB Timeout

**Immediate Actions:**

1. Check database connection.
2. Restart database.
3. Escalate to DBA team.

## Who to Escalate To

*   DevOps Team
*   DBA Team
